# Currículo Interativo & Moderno - Henrique Cavalari

Este projeto é um currículo digital de alta performance, desenvolvido com foco em UX/UI moderno e Cibersegurança. O objetivo foi criar uma identidade visual forte para transição de carreira para a área de Defesa Cibernética/SOC.

## Tecnologias Utilizadas

* HTML5: Estruturação semântica do conteúdo.
* CSS3: Estilização avançada utilizando técnicas de Glassmorphism (efeito de vidro) e Design Responsivo.
* JavaScript: Lógica para interatividade (Barra de progresso de leitura dinâmica).
* FontAwesome: Integração de ícones vetoriais para redes sociais e ações.

## Funcionalidades Principais

* Barra de Progresso Dinâmica: Uma barra no topo da página que acompanha o scroll do usuário, indicando o quanto falta para concluir a leitura.
* Glassmorphism Design: Container translúcido com desfoque de fundo (backdrop-filter), conferindo um visual premium e tecnológico.
* Otimização para Impressão (PDF): Configuração via CSS Media Queries que oculta elementos interativos e otimiza cores para gerar um PDF limpo e profissional ao clicar em "Salvar PDF".
* Botões Interativos: Links diretos para LinkedIn, GitHub e acionamento automático de E-mail via protocolo mailto:.
* Segurança de Navegação: Uso de atributos rel="noopener noreferrer" em links externos para prevenir ataques de tabnabbing.

## Como visualizar o projeto

1. Faça o download ou clone este repositório.
2. Certifique-se de que os arquivos index.html, style.css e script.js estão na mesma pasta.
3. Coloque sua foto no diretório com o nome SUA-FOTO.jpg (ou altere o caminho no HTML).
4. Abra o arquivo index.html em qualquer navegador moderno.

## Foco em Cibersegurança

O projeto reflete minha paixão pela área, utilizando uma paleta de cores sóbria (Deep Blue/Slate) e uma estrutura lógica que prioriza a integridade da informação e a clareza para o recrutador.
